# Triumph Behavioral Website Style Notes

The live site at https://triumphbehavioral.com is served by Squarespace, so there is no editable website source in this workspace. The styling layer in `website/squarespace-custom.css` is intended for Squarespace Custom CSS.

The CSS keeps the public site content and images, but makes the visual language closer to the Triumph Workspace app:

- App-style blue/white palette with the existing gold accent.
- Cleaner fixed header with tab-like navigation and a stronger Contact Us CTA.
- Inter/system typography instead of the current mixed decorative feel.
- White panels, light blue backgrounds, 8px radius, and subtle app-like borders.
- Service offerings converted visually into structured cards.
- More restrained paragraph sizing and mobile spacing.

Small content fixes I noticed while inspecting the live site:

- The page meta description is empty.
- The footer text link labeled `Instagram` points to `http://squarespace.com`; the header Instagram icon points to the correct `https://www.instagram.com/triumph_aba/`.
- The homepage copy is warm, but several sections are long. A tighter hero and shorter service cards would make the professional app-style design feel even stronger.
